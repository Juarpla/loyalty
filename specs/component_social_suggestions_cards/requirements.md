# Requirements: Social Suggestions Cards UI Component

## 1. Feature Description
Renders clean output suggestions with visual prompt suggestions and clipboard handlers in `src/components/social/suggestions-cards.component.tsx`.

## 2. Acceptance Criteria
- **R1:** WHEN rendered, the component SHALL present separate draft copy blocks alongside touch-trigger clipboard copies.
- **R2:** Playwright E2E tests in `tests/e2e/component_social_suggestions_cards.spec.ts` SHALL verify component structure.
